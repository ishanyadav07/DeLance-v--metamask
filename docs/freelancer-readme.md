# Decentralized Freelance Marketplace

> Archived project note migrated from the previous nested folder. Use the files in `docs/` as the source of truth.

A Web3 freelance platform where clients fund jobs through smart-contract escrow, freelancers deliver work in milestones, and disputes are handled by DAO voting.

## Core idea
- No platform commission like Upwork
- Escrow-based payments
- Milestone releases
- Reputation system
- DAO dispute resolution
- Optional NFT-based identity or achievement badges
